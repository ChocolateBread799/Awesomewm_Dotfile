## Setup

```
> yay -S awesome-git platerctl-git picom-ibhagwan-git pavucontrol pulseaudio-ctl

> git clone https://github.com/ChocolateBread799/dotfiles

> cd dotfiles
> mv .zshrc ~/
> mv .config/* ~/.config/
```
