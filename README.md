- [MADE Outer Sans](https://www.behance.net/gallery/87122273/MADE-Outer-Sans-Font)

## Setup

```
> yay -S awesome-git platerctl-git picom-ibhagwan-git

> git clone https://github.com/ChocolateBread799/dotfiles

> cd dotfiles
> mv .zshrc ~/
> mv .config/* ~/.config/
```
